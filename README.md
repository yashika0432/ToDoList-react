# Task Log

A simple and responsive To-Do List application built with React that allows users to add, complete, and delete tasks.
This project demonstrates core React concepts like components, state management, event handling, and conditional rendering.

---

## 🚀 Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑 Delete tasks
- 🔄 Real-time UI updates with React state
- 📱 Responsive design
- ⚡ Fast and lightweight

---

## 🛠 Built With
- React
- JavaScript (ES6+)
- HTML5
- CSS3

---

## ⚙️ Installation
1.Clone the repository
```bash
git clone https://github.com/yourusername/react-todo-list.git
```

2.Navigate to the project folder
```bash
cd react-todo-list
```

3.Install dependencies
```bash
npm install
```

4.Start the development server
```bash
npm start
```
